---------------------Accurate, Dense, and Robust Multi-View Stereopsis-------------------

-Place the jupyter notebook in the same folder as the 'Camera_mat' and 'Images' folder.

-'Images' folder consists of 24 multi-view images.

-'Camera_mat' folder consists of 24 text files containing the camera projection matrix data for each of the images present in the 'Images' folder

-Run all the cells and the parameters of the patches along with the indices and features of the two images used to reconstruct patch will be stored in 'Patches' folder.

-The dataset was downloaded from: https://www.di.ens.fr/pmvs/pmvs-1/index.html
